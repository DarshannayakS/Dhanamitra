# 💰 Dhanamitra  
### AI-Powered Personal Finance Tracker

---

## 📌 Introduction
Dhanamitra is a full-stack, AI-powered personal finance management system designed to help users track, analyze, and optimize their financial activities.

The platform integrates machine learning, financial analytics, and intelligent recommendation systems to provide personalized investment strategies and actionable insights.

---

## 🎯 Key Highlights
- Real-time financial tracking (Income, Expenses, Investments)
- AI-driven investment recommendations
- Peer-based spending analysis
- Goal-oriented financial planning
- Secure and scalable architecture

---

## 🚀 Features

### Authentication & User Profiles
- Secure login and registration
- Personalized financial profiles

### Income Management
- Track multiple income sources
- Historical analysis

### Expense Tracking
- Categorized expense logging
- Visual analytics (charts)
- AI-based insights

### Investment Portfolio
- Track stocks, mutual funds, deposits
- Monitor performance

### Financial Goals
- Set short & long-term goals
- Track progress

### AI Recommendation Engine
- Personalized investment plans
- Monthly savings suggestions
- Feasibility analysis

### Behavioral Analytics
- Peer comparison
- User classification (Saver / Balanced / Overspender)

---

## 🧠 AI & Machine Learning

- Linear Regression for expense prediction
- Compound interest-based forecasting
- Goal feasibility validation
- Data preprocessing and normalization

---

## 🛠️ Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite
- AI/ML: Pandas, NumPy, Scikit-learn
- Version Control: Git & GitHub

---

## 🏗️ Architecture

- Frontend → UI & Visualization
- Backend → Business Logic
- Database → Data Storage
- AI Layer → Insights & Predictions

---

## 🔄 Workflow

1. User registers and sets profile  
2. Inputs financial data  
3. Data stored securely  
4. AI processes behavior  
5. Insights & recommendations generated  
6. Dashboard displays results
